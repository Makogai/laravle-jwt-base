## Base project with JWT set up
#### Installation:
1. `composer install`
2. `php artisan vendor:publish --provider="Tymon\JWTAuth\Providers\LaravelServiceProvider"
   `
3. `php artisan key:generate`
4. `php artisan jwt:secret`
5. `php artisan serve`
